Create a new VST3/AU/AAX plugin with webview UI using the iPlug2 framework which integrates the Neural Amp Modeler DSP https://github.com/sdatkinson/NeuralAmpModelerCore to create an all in one Guitar fx/amp modelling plugin, in the style of https://github.com/sdatkinson/NeuralAmpModelerPlugin.
